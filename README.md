# അതിഥി CET

## About
അതിഥി CET is a hostel discovery and comparison platform developed for students of the College of Engineering Trivandrum (CET). The platform helps students find suitable accommodation near the campus by providing hostel information, facilities, pricing, ratings, and comparison tools in a single place.

The primary objective of the project is to make hostel hunting easier, faster, and more transparent for students who are new to the city or looking for better accommodation options.

## Frontend
The frontend provides an intuitive and student-friendly interface for exploring hostels around CET.

### Features
* Browse available hostels
* Search hostels by name
* Filter hostels by:

  * Gender
  * Budget
  * WiFi
  * Mess
  * Parking
  * Gym
  * Laundry
* Sort hostels by:

  * Distance from CET
  * Rating
  * Rent (Low to High)
  * Rent (High to Low)
* View detailed hostel information
* Compare hostels side-by-side
* Responsive user interface for mobile and desktop devices

## Backend
The backend is responsible for data retrieval, filtering, sorting, and comparison functionality.

### Features
* Hostel listing API
* Search functionality
* Multi-filter support
* Multi-field sorting support
* Pagination
* Hostel details endpoint
* Hostel comparison endpoint
* Integration with Supabase database

### Technologies
* Python
* Flask
* Supabase

## Hostel Comparison
Students can compare two hostels side-by-side based on:

* Hostel Name
* Hostel Type
* Rating
* Lowest Rent
* Distance from CET
* Address
* WiFi
* Mess
* Laundry
* Parking
* Water Bill Included
* Electricity Bill Included
* Vacancies

This feature enables students to make informed accommodation decisions without manually comparing hostel details.

## Purpose
Finding reliable hostel information is often difficult for students joining CET. Hostel details are usually scattered across social media, messaging groups, and word-of-mouth recommendations.

അതിഥി CET aims to provide a centralized platform where students can easily discover, evaluate, and compare hostels near the campus, helping them make better accommodation decisions.
